# About
This project was completed as a collaborative assignment to apply effective design choice, usability principles and modelling behaviours. It is a prototype of an Institute Management System (IMS) that implements Attendance and Timetable screens for student and admin views. Other users and modules have classes and methods defined appropriately as dataclasses. All data is stubbed that can be replaced with a database connection appropriately.
